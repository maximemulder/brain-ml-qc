# Files

This is the list of dataset files that may be downloaded and used for the project. The files should be placed in the `files` directory and not be committed to the repository.

## ABIDE

### `ABIDEII_MRI_Quality_Metrics`

These look to be the automatically generated quality control metrics by MRIQC. Might be useful for comparison.

Link: https://fcon_1000.projects.nitrc.org/indi/abide/abide_II.html

### `ABIDE_manual-assessements`

I expected it to be some textual data given the name, but those are actually some outlinings of the brain structure. Not sure how to interpret it yet.

Link: https://osf.io/haf97/overview

## BAARD

### `BAARD_structural_mri_dictionary.md`

Description of the BAARD MRI dataset. Does not seem very interesting for us.

Link: https://kcniconfluence.camh.ca/spaces/BAAR/pages/145162332/Neuroimaging+Dataset?preview=/145162332/145162336/BAARD_structural_mri_dictionary.md
