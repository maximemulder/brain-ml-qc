# Datasets

This file lists potential datasets for the brain MRI ML QC project.

The main goal is to find a dataset that has annotated non-passing scans.

## Autism Brain Imaging Data Exchange II (ABIDE II)

Link: https://fcon_1000.projects.nitrc.org/indi/abide/abide_II.html
Access: open

### Imaging data download

To download the ABIFDE II imaging data:
1. Create an account to NITRC at this URL: https://www.nitrc.org/account/register.php Creating an account requires clicking a confirmation email but there is no delay or manual verification outside of that.
2. Go to the ABIDE II dataset at this URL: https://fcon_1000.projects.nitrc.org/indi/abide/abide_II.html On the right side, there is a list of institutions that participate in the project, as well as a "Download > Scan Data" link for each institution, simply click that link and the download will start.
3. Each download returns an archive that contains NIfTIs sorted by participants, sessions, and modalities.

### Manual quality assessements

Manual quality assessements from the MRIQC paper are available here: https://osf.io/haf97/overview

### Automated QC metrics

There is an associated QC file `ABIDEII_MRI_Quality_Metrics.zip` that contains CSV files with the following columns for each scan:

`Site_ID,Sub_ID,Session,Series,Unique_ID,CNR,SNR,Qi1,EFC,Smoothness_of_Voxels,FBER,Cortical Contrast`

These quality metrics appear to have been automatically generated by MRIQC.
